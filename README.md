# 🚀 مسبحة - Android APK Builder Project

مرحباً بك! هذا المشروع جاهز بالكامل ليتم تحويله إلى تطبيق APK وتثبيته على هاتفك مباشرة.
تم توليد هذا المشروع خصيصاً لك باستخدام **مُحوّل HTML إلى APK**.

---

## 📱 الخيار الأول: التثبيت السريع وبناء التطبيق عبر GitHub (بدون تثبيت أي برامج على جهازك) - مجاناً تماماً ومستحسن جداً!

قد تظن أنك بحاجة لجهاز كمبيوتر ثقيل لتثبيت Android Studio لتتمكن من بناء التطبيق، ولكن هذا غير صحيح! يمكنك استخدام خدمات GitHub Actions لبناء التطبيق مجاناً في دقيقتين:

1. قم بإنشاء حساب مجاني على موقع **[GitHub](https://github.com)** (إذا لم يكن لديك حساب).
2. أنشئ مستودعاً جديداً (New Repository) باسم مناسب (مثلاً `مسبحة`).
3. اجعل المستودع **العام (Public)** أو **الخاص (Private)** حسب رغبتك.
4. قم برفع كافة محتويات هذا الملف المضغوط (ZIP) إلى المستودع الخاص بك على GitHub.
5. بمجرد رفع الملفات، توجه إلى تبويب **Actions** في صفحة المستودع على GitHub.
6. ستلاحظ بدء عملية بناء تلقائية باسم **"Build Android APK"**.
7. انتظر دقيقتين حتى تكتمل العملية ويظهر لك علامة باللون الأخضر ✅.
8. اضغط على العملية المكتملة، وانزل لأسفل الصفحة في قسم **Artifacts**، ستجد ملف **App-Debug-APK** جاهزاً للتحميل مباشرة على هاتفك وتثبيته!

---

## 💻 الخيار الثاني: البناء المحلي باستخدام جهاز الكمبيوتر الخاص بك

إذا كنت تفضل بناء التطبيق على جهاز الكمبيوتر الخاص بك، اتبع الخطوات التالية:

### المتطلبات المسبقة:
1. تثبيت **Node.js** على جهازك.
2. تثبيت **Java Development Kit (JDK) 17**.
3. تثبيت **Android Studio** مع تفعيل Android SDK.

### خطوات البناء:
1. افتح مبدل الأوامر (Terminal / CMD) في مجلد هذا المشروع.
2. قم بتثبيت Cordova عالمياً إذا لم يكن مفعلاً:
   ```bash
   npm install -g cordova
   ```
3. قم بتثبيت الاعتمادات وإضافة منصة الأندرويد:
   ```bash
   npm install
   cordova platform add android
   ```
4. قم ببناء التطبيق:
   ```bash
   cordova build android --debug
   ```
5. بعد انتهاء البناء، ستجد ملف الـ APK الناتج في هذا المسار:
   `platforms/android/app/build/outputs/apk/debug/app-debug.apk`
